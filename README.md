Version branch to publish and access actual version numbers

## JSON tructure Overview

```json
{
  "ProductName": [
    {"PartVersion1": "Version"},
    {"PartVersion2": "Version"},
  ],
}
```

e.g.

```json
{
  "FillControl": [
    {"FillControl": "1.19.04"},
    {"MachineDescriptionEditor": ""},
    {"VaranUpdater": ""}
  ],
  "QualityMonitor": [
    {"VaranUpdater": ""}
  ]
}
```
